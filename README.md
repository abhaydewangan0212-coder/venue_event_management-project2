# venue_event_management-project


A full-stack web application for online venue booking and event management. Users can book venues, select event packages, order food according to guest count, and calculate the total event bill.

---

## Features

- User Registration & Login
- Venue Booking System
- Event Package Selection
- Food Ordering
- Guest Count Based Billing
- Booking Management
- Admin Dashboard
- Profile Management
- Forgot Password System
- Responsive Design

---

## Tech Stack

### Frontend

- CSS

- JavaScript

### Backend
- PHP

- MySQL
